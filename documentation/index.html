HTML Structure

In this template we used as a framework, the Bootstrap Framework v3.1.1 - http://getbootstrap.com/

The default Bootstrap3 grid system is composed by 12 columns, named like "col-*-1", "col-*-2", "col-*-3"...."col-*-12". Grid systems are used for creating page layouts through a series of rows and columns that house your content. Here's how the Bootstrap3 grid system works:

HTML Markup
<div class="container">
   <div class="row">
      <div class="col-md-12">1 Column - Content Here</div>
   </div>

   <div class="row">
      <div class="col-md-6">1/2 Column - Content Here</div>
   <div class="col-md-6">1/2 Column - Content Here</div>
   </div><div class="row">

   <div class="row">
      <div class="col-md-4">1/3 Column - Content Here</div>
      <div class="col-md-4">1/3 Column - Content Here</div>
      <div class="col-md-4">1/3 Column - Content Here</div>
   </div>
</div>

 

The general template structure is the same throughout the template.

If you need more information, please visit this site: http://getbootstrap.com/css/#grid

How To Edit Styles

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited. For example if you would like to edit the color, open up the file style.css and do the following:

.titlebar h1 {
    color: #000;
}

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your style.css file and make sure that there isn't a similar style that has more weight.

Choosing template layout:

Turn off the animations:

It's easy to turn off the animations you just need to do following:

STEP 1: Open up the index.html file and find initialization code in the footer of the page (before the closing </body> tag). "line: 703" and comment out (delete) this line

   <!-- EXTERNAL SCRIPTS
   ============================================= -->
   ...
   <script src="js/animations.js" type="text/javascript"></script>
   ...

STEP 2: Open up the index.html file find this line of code("line: 28") in <head> section and comment out (delete) this line 

   <!-- On Scroll Animations -->
   <link href="css/animate.css" rel="stylesheet">

Save and see the result!

Scrolling To a Spot :

Scrolling to a spot in an HTML page is pretty simple to do. You have a lot of IDs on the page to scroll to. Write your anchor tags just as you normally would with the href attribute pointed at the ID you would like to move to (e.g. href="#services".) This means that if JavaScript is turned off, your menu will continue to function normally.

Example: 

<nav id="navigation-menu" class="collapse navbar-collapse" role="navigation">
    <ul class="nav navbar-nav navbar-right">
        <li><a id="GoToHome" class="selected-nav" href="#intro">Home</a></li> 
        <li><a id="GoToAbout" href="#about-1">About</a></li> 
        <li><a id="GoToFeatures" href="#features">Features</a></li> 	       
        <li><a id="GoToFaq" href="#faq">FAQ</a></li> 
        <li><a id="GoToClients" href="#clients">Clients</a></li> 
    </ul>
</nav>

 

Initialization Scrolling :

Open up the file custom.js find "line: 187" and add (change) initialization code (menu item ID) :

$("#YourID, #GoToHome, #GoToAbout, #GoToFeatures, #GoToFaq, #GoToClients'" ).scrollTo({ speed: 1400 });

You can offset the location of where you scroll to, control the speed and override the element ID you are scrolling to all through the plugin. For more information please visit this site: http://individual11.github.io/Scroll-To/#demo

 

Contact Form:

This template also use the PHP engine for activate contact process. The message direct send to your email address, so you can edit your email address in contact.php file (this will not be available for anyone to see, it is only used by the server to send your email). To do this, please open contact.php file and change this line: 

$email_to = 'you@youdomain.com';

You can also specify an email subject line (or just leave the one which is there).

$email_subject = 'Your email subject line';

One notice: this will be functional only on your server, php file can not be executed on your local computer.

File Name: contact.php (you must use this filename exactly)

 

Newsletter:

If you want to use the newsletter you need to have a MailChimp account (the funcional Newsletter works only with MailChimp - http://mailchimp.com/).

Once you have the MailChimp account you have to configure the subscription file with your APIKey, ListID and set the Datacenter. Open the file newsletter.php and change the variables:

$apiKey = 'your-api-key'; -How get your Mailchimp API KEY - http://kb.mailchimp.com/article/where-can-i-find-my-api-key

$listId = 'your-list-id'; - How to get your Mailchimp LIST ID - http://kb.mailchimp.com/article/how-can-i-find-my-list-id

$submit_url = "http://us2.api.mailchimp.com/1.3/?method=listSubscribe"; Replace us2 with your actual datacenter - Log into your account and look at the top of your browser. The first part of the URL identifies your datacenter.. - http://status.mailchimp.com/#find-your-datacenter
